# Voca

Voca is an AI-powered career recommendation platform that helps users discover career paths based on their education, interests, skills, preferred industries, and career priorities [file:1].

Unlike job boards, Voca focuses on long-term career guidance by generating 3–5 personalized and explainable career recommendations with clear rationales, skill alignment, and growth insights [file:1].

## Features

- Personalized career recommendations [file:1]
- Explainable match reasons [file:1]
- Skill gap and growth insights [file:1]
- Mobile-first guided assessment flow [file:1]
- Shareable results for mentors or family [file:1]

## Tech Stack

- **Frontend:** React + Material UI [file:1]
- **Backend:** Python with FastAPI or Flask [file:1]
- **Database:** MongoDB [file:1]
- **ML/Recommendation Logic:** Scikit-learn with rule-based MVP logic, expandable to Decision Tree / Random Forest models [file:1]
- **Optional NLP:** Sentence Transformers for free-text interest matching [file:1]
- **Hosting:** Railway or Heroku for MVP deployment [file:1]

## Goal

Voca is built to reduce career confusion and help users make more confident, informed career decisions through structured and transparent recommendations [file:1].
