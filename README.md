# Voca

Voca is an AI-powered career recommendation platform that helps users discover career paths based on their education, interests, skills, preferred industries, and career priorities .

Unlike job boards, Voca focuses on long-term career guidance by generating 3–5 personalized and explainable career recommendations with clear rationales, skill alignment, and growth insights .

## Features

- Personalized career recommendations.
- Explainable match reasons.
- Skill gap and growth insights.
- Mobile-first guided assessment flow.
- Shareable results for mentors or family.

## Tech Stack

- **Frontend:** React + Material UI.
- **Backend:** Python with FastAPI or Flask.
- **Database:** MongoDB.
- **ML/Recommendation Logic:** Scikit-learn with rule-based MVP logic, expandable to Decision Tree / Random Forest models.
- **Optional NLP:** Sentence Transformers for free-text interest matching.
- **Hosting:** Railway or Heroku for MVP deployment.
## Goal

Voca is built to reduce career confusion and help users make more confident, informed career decisions through structured and transparent recommendations.
